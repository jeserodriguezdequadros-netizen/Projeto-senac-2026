#VIAJEI
Projeto criado com a turma de desenvolvimento web no SENAC-RS de POA em 2026/2
O objetivo do projeto **Viajei** é, compartilhar das experiências que vivemos, para que outras pessoas também possam usufruir da mesma experiência. Esse experiências podem ser sobre qualquer tema (livros, séries, filmes, cozinha, esportes, viagens...)

O ***Viajei***, por razões apropriadas é indicado para maiores de 14 anos.
